How Do I Use This?

YoYo is a programming language loosely based on Logo. Since it integrates Java, however, many of the more advanced features require knowledge of Java and how its APIs work.

I've written several different manuals for YoYo:

  • Tutorial: How to get started with YoYo and how to get a project on the web.
  • Command Reference: A list of all of the YoYo commands and what they do.
